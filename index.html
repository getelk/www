Incredibly easy installation

Requirements:

A Linux server with min of 2 cores & 2GB of RAM, a License file (email contact@replicated.com for a demo license), server ports 8800, 9200, 80, 443 accessible from your local machine, SSL cert/key

Get started:

  1. SSH into your Linux server (modern versions of Ubuntu, Debian, Centos, Red Hat & Fedora)
  2. Run this install script
    $ curl -sSL https://install.getelk.com | sudo sh
  3. Access your server via HTTPS on port 8800 & bypass the SSL security warning.
  4. Upload a custom TLS/SSL cert/key or proceed with the provided self-signed pair.
  5. Upload the provided license file (.rli)
  6. Check your email for your license activation code.
  7. Secure your ELK Management console with a password
  8. Configure your ELK instance and click "Save"
  9. Visit the hostname you provided to access ELK

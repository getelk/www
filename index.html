Incredibly easy installation

Get started:

  1. SSH into your Linux VM
  2. Run this install script
    $ curl -sSL https://install.getelk.com | sudo sh
  3. Access your VM on port 8800
  4. Register for notifications of updates
  5. Secure your ELK Management console with a password
  6. Configure your ELK instance and click "Save"
  7. Visit the hostname you provided to access ELK
